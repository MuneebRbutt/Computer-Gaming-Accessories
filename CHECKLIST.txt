╔═══════════════════════════════════════════════════════════════╗
║                      TROUBLESHOOTING                          ║
╚═══════════════════════════════════════════════════════════════╝

Still seeing 404?
→ Double-check Root Directory is set to "nextjs"
→ Check Vercel deployment logs for errors
→ Verify NEXTAUTH_URL has your actual Vercel URL

Images not loading?
→ Verify all ImageKit variables are added
→ Check images exist in ImageKit dashboard
→ Clear browser cache (Ctrl + Shift + R)

Database errors?
→ Verify DATABASE_URL is correct
→ Check MongoDB Atlas Network Access allows 0.0.0.0/0
→ Verify MongoDB user has read/write permissions
